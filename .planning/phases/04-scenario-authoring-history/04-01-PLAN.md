      repositoryId    String

      createdAt       DateTime    @default(now())
      updatedAt       DateTime    @updatedAt

       - repositoryId: z.string().min(1, { error: "Please select a repository" })

    3. Implement `createScenarioAction(prevState: ActionState, formData: FormData): Promise<ActionState>`:

       - On validation failure, return errors

       - Verify repository exists AND belongs to selected app: prisma.repository.findUnique({ where: { id: repositoryId } })
